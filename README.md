This project performs an exploratory analysis of sales data from 2023–2024. It uses R to clean, summarize, and visualize the dataset by city, focusing on average service time and units sold.
The analysis provides a quick understanding of sales performance trends and service efficiency across different locations.

**Key Features**
Data Cleaning – Removes duplicates and handles missing values.

Data Transformation – Converts date fields and groups results by city.

Summary Tables – Calculates:

Average service time per city

Average units sold per city

Visualization – Generates a line chart of monthly sales per city using ggplot2.

**Input Data**
The analysis reads from an Excel file named:
ventas_2023_2024.xlsx
